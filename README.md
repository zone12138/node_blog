# vue_node

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
## Server setup

### startup nodeJs server
```
node ./server/index.js
```
or 

### startup nodeJs server(hot-reloads)
```
nodemon ./server/index.js
```
